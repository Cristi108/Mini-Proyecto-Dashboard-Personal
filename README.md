# Dashboard de Series - Mini Proyecto

Este proyecto es un pequeño análisis de un dataset de series. El objetivo fue crear varias visualizaciones que ayudaran a ver patrones sobre temporadas, episodios, ratings y las plataformas donde se transmitieron.

## Contenido del proyecto
Dentro del notebook se pueden encontrar:

- Carga del dataset de series.
- Limpieza básica y revisión de columnas.
- 5 gráficos de líneas con diferentes colores y estilos.
- Cada gráfico muestra un aspecto distinto del dataset.
- Guardé cada gráfico en formato PNG.
- Incluí algunos insights que encontré después de analizar los datos.

## Gráficos incluidos
1. IMDb Rating por Serie  
2. Rotten Tomatoes por Serie  
3. Número de Temporadas por Serie  
4. Número de Episodios por Serie  
5. Cantidad de Series por Network  

## Herramientas usadas
- Python  
- pandas  
- matplotlib  
- Google Colab
